# Portfolio Artista Plástico

Sitio web minimalista para portfolio de artista plástico con diseño responsivo.

## Estructura

- **Biografía**: Sección con información del artista
- **Obra 1 - Vacío Habitado**: Obra conceptual con descripción
- **Obra 2 - Resonancias Silenciosas**: Obra conceptual con descripción
- **Obra 3 - Tiempo Suspendido**: Obra conceptual con descripción
- **Galería**: Grid de fotos de obras

## Características

- Diseño minimalista y elegante
- Navegación suave entre secciones
- Responsive design para móviles y tablets
- Animaciones sutiles al hacer scroll
- Código limpio y fácil de personalizar

## Personalización

Para añadir tus propias imágenes, simplemente reemplaza los elementos `.placeholder-image` con etiquetas `<img>` que apunten a tus archivos de imagen:

```html
<img src="ruta/a/tu/imagen.jpg" alt="Descripción">
```

## Cómo usar

Abre el archivo `index.html` en tu navegador para ver el sitio web.
